#  Cognifyz Technologies – Machine Learning Internship Project

##  Overview
This repository contains my Machine Learning internship work at **Cognifyz Technologies**, where I worked on a restaurant dataset to perform data analysis, build predictive models, a recommendation system, and classification tasks.

---

## Objective
To apply machine learning techniques on restaurant data to:
- Predict restaurant ratings  
- Build a recommendation system  
- Classify cuisine types based on features  

---

##  Dataset
The dataset includes restaurant-related features such as:
- Cuisine type  
- Ratings and votes  
- Location  
- Cost for two  
- Online delivery availability  

The data was cleaned and preprocessed by handling missing values and encoding categorical variables.

---

##  Tasks Performed

### Task 1: Regression (Rating Prediction)
- Model: Decision Tree Regressor  
- Goal: Predict restaurant ratings  
- Evaluation Metrics:
  - R² Score: 0.9496  
  - RMSE: 0.3385  
  - MSE: 0.1146  

---

### Task 2: Recommendation System
- Method: Content-based filtering  
- Technique: Cosine similarity  
- Goal: Recommend similar restaurants based on feature similarity  

---

###  Task 3: Classification (Cuisine Prediction)
- Model: Random Forest Classifier  
- Goal: Predict cuisine type  
- Evaluation Metrics:
  - Accuracy: 36.85%  
  - F1 Score: 0.32
